## Como instalar

Não existe etapa de instalação obrigatória.

Basta baixar o ZIP, extrair os arquivos e abrir:

```txt
index.html
```

Para rodar em servidor local:

```bash
python -m http.server 8000
```

ou:

```bash
npx serve .
```

3. Public directory: raiz do projeto.

2. Framework: Other.
3. Build command: vazio.
4. Output directory: raiz.

**VittaCare** is a premium healthcare landing page template created for clinics, consultórios, aesthetic centers, dental practices, wellness businesses and health professionals who need a modern and trustworthy online presence.

The template was designed with a strong focus on UX/UI, conversion, accessibility and easy customization. Built with **HTML5, CSS3 and vanilla JavaScript**, VittaCare is lightweight, responsive, mobile-first and easy to adapt for different healthcare niches.

The visual direction combines a clean medical aesthetic, premium spacing, soft gradients, trust-focused copy, elegant rounded cards and smooth microinteractions.

Perfect for designers, freelancers, agencies and developers who want to sell or customize a professional landing page template for healthcare clients.
