1. Visão geral

LivrePro é uma landing page premium criada para profissionais liberais, consultores, advogados, arquitetos, terapeutas, coaches, designers, contadores, personal trainers, educadores e especialistas independentes.

O objetivo do template é apresentar autoridade, diferenciais, serviços, método de trabalho, prova social, planos e formulário de contato em uma estrutura responsiva, moderna e fácil de customizar.

3. Como instalar e abrir

  1. Baixe e descompacte o arquivo ZIP.
  2. Abra a pasta do projeto no editor de código.
  3. Clique duas vezes em index.html para abrir no navegador.
  4. Para desenvolvimento local, use uma extensão como Live Server no VS Code.
profissionais-liberais-landing-template/
└── index.html

4. Como customizar

Trocar textos

Edite diretamente o arquivo index.html. Procure pelos títulos, subtítulos, CTAs, serviços, planos e depoimentos.

Trocar cores

Abra assets/css/style.css e altere as variáveis CSS no início do arquivo:

:root {
  --color-primary: #16b8a6;
  --color-secondary: #4353ff;
  --color-accent: #f7c873;
  --color-bg: #f7f8fc;
  --color-surface: #ffffff;
  --color-text: #0d1b2a;
  --color-muted: #64748b;
  --color-border: #dde3f0;
}

Trocar imagens

Substitua os arquivos dentro de assets/img/. Mantenha nomes iguais ou atualize os caminhos no HTML.

Alterar CTAs

Procure por textos como Agendar diagnóstico, Agendar conversa e Solicitar proposta no index.html.

Duplicar ou remover seções

Cada bloco está organizado como <section class="section">. Para duplicar, copie a seção completa. Para remover, apague o bloco da abertura ao fechamento.

7. Como publicar

GitHub Pages

  1. Crie um repositório no GitHub.
  2. Envie todos os arquivos do template.
  3. Vá em Settings → Pages.
  4. Escolha a branch principal e publique.

Netlify

  1. Acesse o painel do Netlify.
  2. Arraste a pasta do projeto para deploy.
  3. O site será publicado automaticamente.

Vercel

  1. Importe o repositório.
  2. Como é um site estático, não precisa configurar build.
  3. Publique diretamente.

Render

  1. Crie um Static Site.
  2. Conecte o repositório.
  3. Use a pasta raiz como publicação.
