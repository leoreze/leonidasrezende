## How to Use

Open the project folder and run a simple local server:

```bash
python -m http.server 8080
```

Then open:

```txt
http://localhost:8080
```

You can also open `index.html` directly in a browser.

## How to Customize

### Change colors

Edit the CSS variables at the top of:

```txt
assets/css/style.css
```

Example:

```css
:root {
  --color-primary: #2DD4BF;
  --color-secondary: #38BDF8;
  --color-accent: #F59E0B;
  --color-bg: #080D18;
  --color-surface: #0B1220;
  --color-text: #F8FAFC;
}
```

### Change logo

Replace:

```txt
assets/img/logo-mark.svg
```

You can use SVG, PNG or WebP. Update the `<img>` path in `index.html` if the filename changes.

### Change text content

All titles, labels, navigation items, metrics and cards are inside:

```txt
index.html
```

### Change chart data

Edit the `chartData` object inside:

```txt
assets/js/main.js
```

### Remove sections

Each major dashboard area is separated by semantic `<section>` blocks. You can remove or reorder these blocks in `index.html`.

## Deployment

### GitHub Pages

1. Create a GitHub repository.
2. Upload the project.
3. Go to Settings → Pages.
4. Select the main branch and root folder.
5. Publish.

### Netlify

1. Drag and drop the folder into Netlify.
2. No build command required.
3. Publish directory: project root.

### Vercel

1. Import the repository.
2. Keep build command empty.
3. Set output directory to root if needed.

### Render

Use Static Site deployment and set the publish directory to the root folder.
