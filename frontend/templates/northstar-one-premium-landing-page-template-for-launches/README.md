## How to use

1. Download and unzip the package.
2. Open `index.html` in your browser.
3. Edit text, CTAs and links in `index.html`.
4. Edit colors and visual tokens in `assets/css/style.css`.
5. Edit interactions in `assets/js/main.js`.
6. Publish as a static website.

## How to change colors

Open `assets/css/style.css` and edit the variables:

```css
:root {
  --color-primary: #0D1321;
  --color-secondary: #1F6FEB;
  --color-accent: #D6B268;
  --color-bg: #F4EFE6;
  --color-surface: #FFFFFF;
  --color-text: #0D1321;
  --color-muted: #5F6B7C;
  --color-border: #DED4C5;
}
```

## How to edit CTAs

Search inside `index.html` for:

- `Get the template`
- `Comprar template`
- `Ver preview`
- `Explore sections`
- `Enviar interesse`

Replace the text or change the `href` attributes.

## How to duplicate a section

Copy any `<section>` block in `index.html`, paste it where needed, and keep the `.container` class for consistent spacing.

## How to remove a section

Delete the full `<section>` block and remove the related menu link in the header if necessary.
